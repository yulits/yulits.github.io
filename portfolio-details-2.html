Wine Clustering Project

Project information

  • Category: Unsupervised learning
  • Project date: 2020
  • Read and execute

In this project I compare performence of different clustering algorithm, such as KMeans, Affinity Propagation, Agglomerative Clustering, Birch, MeanShift and DBSCAN, using different metrics (ARI, Silhouette Coefficient, Homogeneity, Completeness and V-Measure).
